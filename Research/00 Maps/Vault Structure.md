# Vault Structure

This vault uses exactly one canonical folder for each content type:

- `01 Literature Notes` for paper notes
- `02 Concepts` for connected concept notes
- `03 Glossary` for glossary access
- `04 Equations` for sourced equations
- `05 References` for bibliographic data
- `06 Experiments` for experimental records
- `07 Predictions` for falsifiable predictions
- `08 Derivations` for derivation chains
- `09 Full Text` for extracted paper text
- `10 PDFs` for source files
- `11 Collections` for research branches

Do not keep parallel unnumbered `Concepts` or `Literature Notes` folders.
