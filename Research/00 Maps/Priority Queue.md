# Priority Queue

Priorities are ranked by expected reduction of scientific uncertainty, not by ease of publication or visual appeal.

## P0 — Empirical closure

### Photodiode harmonic-state experiment

- [ ] Freeze the prospective protocol and decision thresholds.
- [ ] Record device identifiers, source wavelengths, angular modulation, exposure duration, environmental conditions, and acquisition settings.
- [ ] Define blinded controls, sham exposures, dark controls, thermal controls, and device-swap controls.
- [ ] Preserve raw oscilloscope and logger files with checksums.
- [ ] Run null-model, harmonic, persistence, and multiple-comparison analyses.
- [ ] Separate discovery data from confirmation data.
- [ ] Obtain independent apparatus replication.

### Water-cavity resonance

- [ ] Reconstruct the complete apparatus and acquisition chain.
- [ ] Separate optical, acoustic, thermal, mechanical, electronic, and software artifacts.
- [ ] Recalculate ringdown and effective $Q$ with uncertainty propagation.
- [ ] Repeat using independent cavities, detectors, operators, and analysis code.

## P1 — Mathematical closure

### Core action to field equation

- [ ] Define the canonical action and regularization domain.
- [ ] Perform the complete higher-derivative variation.
- [ ] State boundary conditions and admissible function spaces.
- [ ] Derive conserved currents and stress-energy consistently.
- [ ] Establish well-posedness or clearly delimit the unresolved regime.

### Finite-$k$ selection and Lyapunov descent

- [ ] Prove the dispersion maximum and unstable band.
- [ ] Derive the Lyapunov functional from the chosen dynamics.
- [ ] State the conditions under which spectral entropy decreases.
- [ ] Distinguish heuristic support shrinkage from theorem-level results.

### Confinement to mass

- [ ] Derive the loop-locking Euler–Lagrange equation.
- [ ] Prove when $k_{\mathrm{eff}}=\langle\sigma\rangle_w$ is valid.
- [ ] Check dimensional consistency and covariance.
- [ ] Derive the relation to $E=mc^2$ without circular identification.

### Spatial dimensionality

- [ ] Separate Sobolev embedding facts from the stronger physical claim.
- [ ] State the exact nonlinear PDE and equilibrium class.
- [ ] Prove or withdraw the claim that all nontrivial equilibria are unstable for $n\ge4$.

## P4 — Recognition and preservation

- [ ] Complete DOI, ORCID, DataCite, OpenAlex, Crossref, GitHub, and Zenodo linkage.
- [ ] Maintain paper-to-paper dependency and chronology maps.
- [ ] Record independent citations, forks, replications, and substantive discussions.
- [ ] Preserve exact release dates, versions, equations, predictions, and code commits.

## Ranking rule

For each task, estimate

$$
R=\frac{U\,F\,I}{T},
$$

where:

- $U$ is uncertainty reduced;
- $F$ is falsifiability gained;
- $I$ is independence from the original analysis;
- $T$ is time and resource cost.

Work the highest-$R$ task that is currently feasible.
