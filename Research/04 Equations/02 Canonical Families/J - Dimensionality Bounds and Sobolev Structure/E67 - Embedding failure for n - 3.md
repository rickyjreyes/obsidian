# E67 — Embedding failure for $n > 3$

> [!info] Classification
> **Family:** J - Dimensionality Bounds and Sobolev Structure  
> **Status:** canonical

## Canonical statement

Counterexample existence:

$$\exists\, \psi \in H^2(\mathbb{R}^n) : \psi \notin L^\infty(\mathbb{R}^n) \quad \text{if } n > 3$$
