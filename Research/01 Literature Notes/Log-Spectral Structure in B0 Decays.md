## Abstract / record description

This record presents an independent diagnostic analysis of the LHCb open-data B⁰ → K*⁰ μ⁺μ⁻ candidate yield spectrum. The study tests whether the q² distribution contains reproducible log-periodic residual structure beyond a smooth baseline, using KDE baseline repair, widened charmonium vetoes, bounded Poisson likelihood tests, null comparisons, integer active-domain winding, Koide-comb geometry, sideband checks, and veto-window covariance tests.

The tested residual form is:

ℓ = ln(q²)

with bounded log-periodic component:

Δ(ℓ) ∼ A · cos(kℓ + φ)

where:

• A - bounded residual amplitude• k - log-frequency• φ - phase offset• ℓ - logarithmic momentum-transfer coordinate

After replacing the initial Savitzky–Golay baseline with a resonance-aware KDE baseline, a high-k residual family survives across the KDE bandwidth ladder:

s_KDE ∈ [0.50, 1.50]

The free-frequency optimum shifts with KDE bandwidth, while the structure maps more naturally into active-domain winding coordinates:

n = k Δℓ_A / (2π)

where Δℓ_A is the retained active log-domain support after vetoes.

In this coordinate, the reference branch maps near the stable integer active-domain winding:

n_ℓ = 15

The integer-winding family:

{10, 15, 20}

encodes the charged-lepton Koide ratio:

Q_ℓ = 2/3

as a true-sideband winding ratio.

In the unrestricted trigonometric scan, the folded Q = 4/9 branch gives a stronger ΔD, while the true-sideband Koide branch remains the Class I reference geometry. A well-first scan, performed before imposing the comb model, finds the cleanest raw Koide-like triplet in the B-low sideband:

(9.8141, 14.5614, 19.0196)

with:

Q_mean ≃ 0.6635

The signal-window triplet:

(12.1421, 17.6350, 23.5082)

is related to the B-low triplet by a resolved-regime dilation:

n_sig ≃ 1.2283 n_Blow

with:

p_scale = 0.00586

under random triplet pairing in the resolved-well regime.

The scale factor also lies close to the candidate inverse-Koide square-root dilation:

a ≃ 1.228287√(Q_ℓ⁻¹) = √(3/2) ≃ 1.224745

a difference of approximately 0.29%.

The result supports a structured candidate-spectrum log-domain with active-domain winding stability, Koide-like sideband geometry, and scale-coupled signal/sideband projections. More broadly, the work introduces log-spectral and active-domain winding diagnostics as tools for identifying structured residual geometry in collider candidate spectra.
