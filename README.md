# WCT Obsidian Research Vault

This repository is the version-controlled knowledge system for Richard J. Reyes's Wave Confinement Theory research program.

## Main goal

Build a permanent, auditable research graph in which every major WCT claim is connected to:

1. plain definitions and assumptions;
2. a complete derivation chain;
3. canonical equations and symbol definitions;
4. numerical implementations and reproducible parameters;
5. a quantitative prediction;
6. an experiment or public-data test;
7. an evidence status and uncertainty statement;
8. an explicit falsifier;
9. the papers, releases, code, and chronology establishing priority.

The vault is not merely a paper archive. It is intended to function as a proof, experiment, and research-priority engine.

## Primary entry point

Open:

- `Research/00 Maps/WCT Research Command Center.md`

## Core layers

- `Research/00 Maps/` — command center, dependency maps, chronology, and indexes
- `Research/01 Literature Notes/` — one note per paper
- `Research/02 Concepts/` — canonical glossary and concept graph
- `Research/03 Equations/` — one note per equation and equation family
- `Research/04 Derivations/` — stepwise proofs and reduction chains
- `Research/05 Predictions/` — measurable predictions and falsifiers
- `Research/06 Experiments/` — protocols, controls, raw-data links, and analyses
- `Research/07 Evidence/` — corroboration, replications, prior art, and claim audits
- `Research/08 Projects/` — active simulations, code, and engineering work
- `Templates/` — note templates enforcing the research schema

## Research standard

A claim is not marked closed merely because it appears in a paper. Closure requires traceability from assumptions to derivation to observable consequence and an evidence record that distinguishes:

- derived;
- simulated;
- predicted;
- observed;
- independently replicated;
- falsified;
- unresolved.
